## 2. Опис реалізації

### Ієрархія класів
Для автоматизації створення коду (гетерів та конструкторів) використано бібліотеку **Lombok**.

* **`Tariff` (Abstract)**: Базовий абстрактний клас, що містить спільні поля: назва, ГБ інтернету, хвилини та ціна.
* **`BasicTariff`**: Стандартний тарифний план. Містить ім'я абонента та базовий набір послуг.
* **`BusinessTariff`**: Тариф для корпоративних клієнтів. Дозволяє підключати від 1 до 3 пристроїв (реалізована валідація через сетер).
* **`RoamingTariff`**: Спеціалізований тариф, що включає додаткові ліміти на дзвінки та інтернет за кордоном.
* **`PotushniiTariff`**: Преміальний план з необмеженими ресурсами (`POSITIVE_INFINITY`).

### Функціонал програми (`Main.java`)
1.  **Вивід:** виводить всіх абонентів 
2. **Сортування:** Об'єкти зберігаються у `List<Tariff>` та сортуються за допомогою `Comparator.comparingDouble(Tariff::getPrice)`.
3. **Фільтрація:** Користувач вводить межі ціни, після чого метод `searchSuitableTariff` відбирає відповідні об'єкти.
